Visualization Story: Children Living in Poverty


STATIC

by Brittney Younger

This visualization originated from trying to find a better way to analyze multiple variables against one another. The original graphic was a table that showed 16 indicators with two years’ worth of data per racial group. One of the main challenges was to capture a way of comparing each racial group for each indicator without over cluttering the designated space and trying to find the right way to tell a story through the data.





Using a horizontal dot plot format, I decided to focus on just one indicator for this project. Using Excel, I created each graph by flipping a line graph horizontally, adding markings and removing the line. The dot plot provides a comparison between each racial group.
