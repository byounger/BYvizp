Visualization Story: Back-to-Back Comparison


INTERACTION

by Brittney Younger

Similar to the Treemap Visualization the following comparison table was developed for a mobile application prototype. I wanted to create an easy to use visualization that allows users to compare Washington D.C.'s neighborhoods against each other. I wanted the user experience to be somewhat like using a mobile sport application. Where a neighborhood statistics faced off against each other. Each indicator group is color coded and the years designated by the color opacity.


Users can select both drop-downs to compare up to 39 cluster groups that represent D.C. neighborhoods.
